# CSP-J 模拟考试系统

CCF 非专业级软件能力认证入门级（CSP-J）在线模拟测试系统，基于 Python Flask 构建。

## 功能特性

- **随机组卷**：从题库中随机抽取指定数量的单选题，每次题目顺序随机
- **即时批改**：提交后立即显示正确率、正确答案和详细解析
- **知识点标签**：每道题自动归类到"数制与编码"、"数据结构"、"算法"、"C++程序设计"、"计算机网络"、"计算机基础与系统"、"数学与逻辑"等知识领域
- **成绩历史**：所有考试记录保存到 SQLite 数据库，可随时回顾
- **错题重练**：针对答错的题目进行专项练习，支持按学生筛选
- **排行榜**：单场最佳成绩排行 + 累计答对总题数排行
- **学生管理**：支持多学生使用，每人独立的考试记录和错题集

## 快速开始

```bash
# 1. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动服务
python app.py
```

浏览器打开 http://localhost:8080 即可使用。

## 项目结构

```
csp_mn/
├── app.py              # Flask 主应用（路由、组卷、批改）
├── db.py               # SQLite 数据库层（考试记录、排行榜）
├── questions.py        # 题库（180 道题）
├── requirements.txt    # Python 依赖
├── exam.db             # SQLite 数据库文件（自动创建）
├── static/
│   └── style.css       # 样式
└── templates/
    ├── index.html      # 首页（组卷、错题重练入口）
    ├── quiz.html       # 答题页
    ├── result.html     # 成绩单（答案解析）
    ├── history.html    # 成绩历史
    ├── exam_detail.html# 考试详情回顾
    └── leaderboard.html# 排行榜
```
